Version 1.3.0
- Android (first version)
- Add icons
- Switch licence to PolyForm Noncommercial License 
- Click/drag in UI
- Logging level can be set/loaded from the settings
- Add shadow blur multiplier setting
- Most library dependencies are pulled in through git sub-modules
- Add a night-time skybox (Qwantani Moon)

Version 1.2.0
- Support parallax mapping in VR for panoramic scenes
- Add the Bedlam model (thanks Richard West)
- Add an indoor version of the Spitfire
- Add the Sopwith Pup
- Soft shadows
- Reduce glider launch speeds
- Remove support for OpenGL 1
- Remove support for stereoscopic display (it was a poor substitute for VR)
- Fix crash with circular runways but not a control-line plane

Version 1.1.0
- VR support (tested on Oculus Rift)

Version 1.0.1
- Fix logging location - will be %APPDATA%/Rowlhouse/PicaSim/Logs/
- Add a PlayStation controller mapping (thanks 8vD!)
- Support for joysticks (i.e. R/C transmitter interfaces) as well as gamepads
- Fix opening joystick calibration
- Support "copy" in PropellerEngine definitions. Tweak quads.
- Fix toggling of fullscreen when pressing F during text input (saving)
- Hopefully fixUI/ OpenGL errors

Version 1.0.0
- Complete port to SDL2 and OpenGL for Windows
- New ImGui-based menu system replacing IwUI
- OpenAL-Soft audio with 3D positional sound
- SDL2 input handling (keyboard, mouse, gamepad)
- CMake build system with vcpkg dependency management
